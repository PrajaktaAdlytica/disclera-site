# Disclera Logo Assets

- `disclera-logo-primary.svg`: primary horizontal logo. `dis` is Espresso and `clera` is Sage.
- `disclera-logo-monochrome.svg`: single-color horizontal logo.
- `disclera-logo-reverse.svg`: logo for dark backgrounds.
- `disclera-mark.svg`: standalone index-page monogram.
- `disclera-favicon.svg`: square favicon with optical padding.

## Palette

- Espresso: `#25352D`
- Sage: `#66806A`
- Pistachio: `#DCE8C8`
- Paper: `#FAF7F1`

The production vectors are traced from the approved option 3 artwork. The only
intentional color adjustment is Sage on `clera`. All letterforms are outlined,
so the SVGs have no runtime font dependency.
